# manage-apps

Get Started

- Technologies I used for this project:

1. Front-end (ReactJS).
2. Back-end (NodeJS)
3. Database: (Firebase)
4. Send and receive SMS: ( Twilio ).
5. Source version control: (Github)

- How to work:

* run frontend:
  yarn install
  yarn start

* run backend:
  yarn install
  yarn start
